Traditional Mongol Script

This is the historical vertical script (Mongol bichig), written top-to-bottom in columns. In this app, it is the target script you learn to read and write.

Cyrillic Mongolian

Modern Mongolian in Mongolia is commonly written in Cyrillic. Many learners already know this system, so it is used as a practical bridge into Traditional script.

Latin Input

Latin is included as an accessibility/input layer (for keyboards and beginners). It is not the final writing system, but it helps users start practice quickly.

Why Taught This Way

The learning flow moves from familiar input systems (Cyrillic/Latin) to Traditional script output, then from recognition to production. This reduces beginner overload and improves retention.

  • Start with transliteration and letter recognition in Studio.
  • Move to Script Writing Quiz for typed and freehand output practice.
  • Use Lessons and Cheat Sheet for structure, rules, and quick reference.

Transliteration Note

The built-in transliteration is currently rule-based and deterministic, designed for learning clarity. It is not a full linguistic parser for every lexical/orthographic edge case.

  • Great for core learning and script writing practice.
  • Ambiguous words can still require context or dictionary support.
  • Best future upgrade: hybrid model (rules + exceptions dictionary).

Tip: install a Mongolian-capable font (such as Noto Sans Mongolian) for best rendering quality.
